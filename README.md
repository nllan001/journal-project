journal-project
===============

cs122a project

Connections:
LCD:
1 - Gnd
2 - Vcc
3 - Gnd through resistor
4 - Reset at Leader D2
5 - Gnd
6 - Enable at Leader D3
7-14 - PORTB

Joystick:
GND - Gnd
U/D - A1
L/R - A0
L/R+ - Vcc
U/D+ - Vcc

Photoresistors:
Module 1:
Left - A2
Up - A3
Right - A4
Down - A5
